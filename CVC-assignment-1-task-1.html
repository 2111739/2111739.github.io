Unit 1: Section 1 & Project Ideas
Joshua Ofosu Asare 2111739

BEng Embedded Electronic Systems

The chosen project idea is "Accelerometer and Gyroscope assisted RC controller". There is already some research out there into controlling wheelchairs [4], and simple implementations for RC Cars [3]. The goal of this project is to create a framework which makes it easier to integrate this technology across different platforms. It is vital to keep in mind the new technologies that are appearing and integrate those into this project where possible. This is a project which could benefit from both wearable technology and IoT connectivity. This project can make a variety of technology more accessible to groups who would otherwise typically struggle to reap the full benefits of RC-based technology, as well as adding extra user throughput through the system.

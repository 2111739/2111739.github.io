Research Philosophy

Our chosen research philosophy is Critical Realism which assumes “a scientific approach to the development of knowledge but is nuanced from positivism as it does not require the restrictions of science to be proven. Knowledge is real if it can be experienced by the senses as exemplified by the phrase ‘what you see is what you get’, and it is then retrospectively explained using thought. However, critical realists also accept that we can be filled by things like optical illusions, as they understand that there is a bigger picture in play” [1]. (UWTSD, 2022). This approach was chosen as the project relies on using sensors to improve the human experience of using an RC Controller. In essence, the project is trying to trick the user into believing that the Controller and RC device are the same by instructing the RC device to react to the rotation of the RC controller as if the user was rotating the device itself.

Known Knowns

A few pieces of knowledge that everyone can be assumed to be aware of and certain about [1] are the general principles of an RC controller – you change the inputs of the controller and you expect to see a change in the output of the device being controlled. Another aim of this research project is to attempt to exploit the fact that there are known knowns and known unknowns – it can be assumed that the users do not understand how RC controllers operate, to make them feel directly linked to the device under control. The databases that will be utilised for this research are: the UWTSD Library, Google Scholar, Google, YouTube. The Library and Google Scholar will be used to find academic sources of information for this project. Google and YouTube will both be used to find existing implementations of similar ideas if they already exist. Then they will be contrasted with this project to ensure that this project is able to provide value by generating knowledge that wasn’t known before. Boolean search strategies will be employed in order to obtain specific search results. AND in particular will be used to return search results that have match several key words. As this project is looking at RC controller and Gyroscopes/Accelerometers, and example search would look like: RC Controller AND (Gyroscope OR Accelerometer).

Research Strategies

The strategy for this project is to first clarify the research and identify the main concept of the research [3] [4]. Secondly data must be collected and a number of options are available: Surveys, Focus Groups, Interview, Scanner Panels, Web Tracking, Ethnography, Experimental Research [5]. For this project, Surveys will be the chosen method of data collection for this project as it will allow us to heavily tailor the questions we ask of users to receive specific answers.
